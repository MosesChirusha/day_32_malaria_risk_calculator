A lightweight, browser-based health awareness tool designed to help individuals, schools, NGOs, and community health workers assess malaria risk factors and learn practical prevention steps. Built to work in low-connectivity environments using only HTML, CSS, and vanilla JavaScript.

---

## Overview

This project is part of the **90 Days of African Solutions Challenge**, which focuses on creating simple, impactful digital tools for real-world problems across Africa.

**Problem**  
Low awareness of everyday malaria risk factors leads to delayed prevention and treatment, especially in rural and high-risk areas.

**Solution**  
An interactive risk assessment tool that evaluates environmental and behavioral factors and presents users with a clear risk level and tailored prevention tips.

## Tech Stack

- HTML  
- CSS  
- Vanilla JavaScript  

No frameworks, build tools, or external dependencies required.

---

## Features

- Simple, mobile-friendly interface  
- Transparent risk scoring logic  
- Instant risk level feedback  
- Tailored malaria prevention tips  
- Works offline once loaded  
- Suitable for schools, clinics, and NGO deployments
